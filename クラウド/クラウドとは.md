**クラウド** : いつどこでもインターネット回線を通じてアクセスできる環境

クラウドのサービス提供の仕方

**SaaS(サース)** : アプリ + OS + インフラ ex) SNSやDropbox

**PaaS(パース)** : OS + インフラ

**IaaS(イアース)** : インフラ　ex) ネットワークやサーバのマシン

サーバ : 何かサービスを提供するもの。「⚪︎⚪︎サーバ」の機能は、**ソフトウェア**で提供される。OSの上でソフトウェアが動く。
　　　　　　　　　　　　　　　　一つのコンピュータにサーバの上限などないため、複数のソフトウェアをインストールすれば、一つのPCで複数のサーバの役割を担える。

PCを構成する要素
CPU : パソコンの頭脳。制御・演算などの処理をしている。プログラム実行はCPUが担当。
メモリ(メインメモリ) : 一時的な記憶装置。CPUは記憶できないので、データの格納場所として使う。
マザーボード : 
ストレージ : 補助記憶装置。HDDやSSD。メモリに書き込んだ内容は、電源を落とすと消えてしまうため、永続的に残したデータは、ストレージに書き込む。
OS : コンピュータを動かすためのシステム。ハードウェアと、OS上で動くソフトウェアの仲介をしている。
